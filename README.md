# Humanoid Robotics Network — Website v1

Static website for GitHub Pages.

## Publish on GitHub Pages

1. Create the repository `website` inside the `Humanoid-Robotics-Network` organization.
2. Upload all files from this folder into the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/ root`.
6. Save.

The site should become available at:

`https://humanoid-robotics-network.github.io/website/`

## Editing

- Edit text directly in `index.html`.
- Replace images inside `assets/`.
- Design settings are in `style.css`.
